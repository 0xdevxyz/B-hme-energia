# Business Model – Studio

## Revenue Streams

### 1. Setup Fee (Einmalig)
What's included:
- config.json setup
- Generation of main index.html
- deployment
- 1x Magic Link test

Suggested pricing: €299–€599 depending on complexity

### 2. Monthly Hosting (Abo)
What's included:
- Server hosting of editor backend
- GEO/SEO template updates via `studio update`
- Basic support (1 edit request/month via email)

Suggested pricing: €29–€49/month

### 3. Landing Pages (On-demand)
What's included:
- One additional local or service landing page
- Added to sitemap + llms.txt
- Keyword research for the page

Suggested pricing: €99–€199 per page

### 4. Content Updates
What's included:
- You make edits on behalf of client
- Or: client uses Magic Link overlay themselves (included in hosting)

Suggested pricing: €49/hour or included in premium hosting tier

## Packages

| Package | Price | Includes |
|---|---|---|
| Starter | €299 setup + €29/mo | 1 main page, hosting, updates |
| Growth | €499 setup + €49/mo | 1 main + 3 landing pages, hosting, updates, priority support |
| Pro | €799 setup + €79/mo | 1 main + 10 landing pages, hosting, updates, monthly report |

## Upsell Opportunities
- Additional landing pages: €99–€199 each
- Google Business Profile setup: €99 one-time
- Monthly SEO report: €49/month
- Social media content bundle: custom pricing
- Blog/news section: €299 setup

## Client Lifetime Value
- Average client: 12 months × €39 = €468 + €399 setup = **€867 LTV**
- Growth client: 24 months × €49 = €1,176 + €499 setup + 5 extra pages = **€2,175 LTV**
